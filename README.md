# BooksKeep 
Ecommerce website for books

- Project Link : https://bookskeep.netlify.app/

## Features

1. Home page
2. Product listing page
3. Filter and sort products
4. Search product
5. Individual product page
6. Cart Managment
7. Wishlist managment
8. Address Managment
9. Checkout page
10. Authentication:
11. User Signup
12. User Login
13. Fully Responsive

### Build With 

React JS
React Context API + useReducer (State Management)
React Router
CSS 
Firebase (Backend)
